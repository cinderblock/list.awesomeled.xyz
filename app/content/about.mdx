## History

Cameron Tacklind started this project as a Google Sheets document to organize information about LED controllers, pixels, and related products.
With the help of communities like [LEDs ARE AWESOME](https://www.facebook.com/groups/LEDSAREAWESOME) on Facebook and [r/led](https://www.reddit.com/r/led/) and [r/WLED](https://www.reddit.com/r/WLED/) on Reddit, contributors added more data and tabs over time.

As usage grew, Google Sheets' limitations and drawbacks became apparent.
Cameron had envisioned a dedicated website from the beginning but never found the time.
In 2026, Claude built the initial website, and Cameron refined it into the first public version.

Now we invite the community to help keep the data accurate and up-to-date.

## Contributing

All data is stored in human-readable YAML files on GitHub.
If you'd like to add or update information, you can submit a pull request to the repository.

## Disclaimer

Information is provided "as-is" with no guarantee of accuracy.
Product specifications may change, and we recommend verifying details with manufacturers before making purchasing decisions.
