Microcontroller development boards are the foundation of DIY LED projects.
These small computers can run firmware like WLED, ESPixelStick, or custom code to control your pixels.

Popular choices vary by connectivity and processing power:

- **ESP32/ESP8266** - WiFi-enabled, great for WLED projects
- **Teensy** - High-speed GPIO for demanding applications
- **RP2040** - Flexible PIO for precise timing
- **Arduino** - Beginner-friendly with extensive library support
